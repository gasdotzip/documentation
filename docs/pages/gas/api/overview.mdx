# API Overview

| Route                                   | Description                                                          |
| --------------------------------------- | -------------------------------------------------------------------- |
| [Chains](/gas/api/chains)               | Returns available chains                                             |
| [User](/gas/api/user)                   | Returns a user's usage history                                       |
| [Deposit](/gas/api/deposit)             | Returns deposit status and associated outbound txs                   |
| [Outbound](/gas/api/outbound)           | Returns outbound tx details and related deposit info                 |
| [Search](/gas/api/search)               | Combined search endpoint for both /deposit and /outbound             |
| [Quote](/gas/api/quote)                 | Returns expected output from bridging                                |
| [Quote Reverse](/gas/api/quote-reverse) | Returns required deposit amount for a given output                   |
| [Call Data](/gas/api/call-data)         | Returns ready-to-use calldata for a given deposit and outbound chain |
