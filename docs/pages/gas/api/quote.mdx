## Quote

> This route gives a live quote returning the amount received on the given outbound chains.

```
https://backend.gas.zip/v2/quotes/<deposit_chain>/<deposit_wei>/<outbound_chains>
```

<Callout type="warning" className="note">
  All chain IDs on this route use the native id not the Gas.zip ID.
</Callout>

<Callout type="info" className="info">
  The outbound chains parameter is a comma separated string
Example: backend.gas.zip/v2/quotes/1/74176868561679620/7777777,81457,3776,666666666

</Callout>
