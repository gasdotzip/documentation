## Rip.Fun Token Deposit

`0x773fC66C488589379F72793f343f6329d10028Bf` is the alternate deposit address that will trigger a `Base USDC` send to the address that sent to it.

<Callout type="warning" className="note">
For ERC20 tokens, transfer using any method tokens to the deposit address. The sender will be the recipient of the Base USDC.

For Native tokens, transfers MUST be a top-level EOA->EOA transfer between the desired recipient, and the deposit address. Calldata will be ignored.
</Callout>

Deposit Flow:

1. Use the [Quote Token API](/gas/api/quote-token) to receive live output quote amount and to ensure sufficient liquidity
2. Send transaction to deposit address using given method
3. Poll [Deposit API](/gas/api/deposit) using transaction hash
4. Wait until Deposit 'status' response is CONFIRMED
