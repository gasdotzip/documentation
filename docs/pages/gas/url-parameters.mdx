# URL Parameters

Gas.zip supports URL parameters for quick access to specific chain and amount configurations:

```
https://www.gas.zip/?chainIds=CHAIN_IDS&dollarAmount=AMOUNT
```
Parameters:
- `chainIds`: Comma-separated list of chain IDs (e.g., `43114,10` for Avalanche and Optimism)
- `dollarAmount`: Dollar amount per chain (e.g., `2` for $2 per chain)

<Callout type='info' className="note">
Example URLs:
- Single chain: `https://www.gas.zip/?chainIds=43114&dollarAmount=2` (Avalanche, $2)
- Multiple chains: `https://www.gas.zip/?chainIds=43114,10&dollarAmount=2` (Avalanche and Optimism, $2 each)

You can get Gas.zip's supported chain IDs from the [Chain Support](/gas/chain-support/outbound) page.
</Callout>
